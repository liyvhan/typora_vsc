# 链接跳转测试

## 外部链接
- 按住 **Ctrl** (Windows/Linux) 或 **Cmd** (Mac) 并点击：[VS Code 官网](https://code.visualstudio.com/)
- 按住 **Ctrl/Cmd** 并点击：[Baidu](https://www.baidu.com)

## 本地文件跳转
- 按住 **Ctrl/Cmd** 并点击跳转到基础测试：[基础测试 (basic.md)](./basic.md)
- 按住 **Ctrl/Cmd** 并点击跳转到代码测试：[代码测试 (code.md)](./code.md)

## 提示
由于这是所见即所得模式，直接点击链接会进入编辑状态。为了方便跳转，我们模仿了 Typora 的行为：**必须配合修饰键 (Ctrl 或 Cmd) 点击** 才能触发跳转。
